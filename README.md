 Netflix Dashboard Analysis (Power BI)

 Project Overview
This project analyzes Netflix dataset using Power BI. It provides insights into content distribution, genres, countries, and yearly trends.

 Features
- KPI Cards (Total Titles, Movies, TV Shows)
- Content Distribution (Pie Chart)
- Year-wise Growth (Line Chart)
- Top Countries & Genres
- Interactive Filters (Slicers)

 Tools Used
- Power BI
- Power Query
- CSV Dataset
